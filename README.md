# IMV Customer Segmentation with K-Means

This repository contains an individual Machine Learning assignment for IMV Recruitment 2026.

## Project Topic

Customer segmentation based on shopping behavior using K-Means Clustering.

## Project Overview

This project aims to segment e-commerce customers based on behavioral patterns such as recency, frequency, monetary value, payment behavior, delivery experience, review score, and product diversity.

The project uses the Olist Brazilian E-Commerce Public Dataset and builds a customer-level analytical dataset before applying K-Means Clustering.

## Objectives

- Understand the raw Olist e-commerce tables.
- Clean and prepare transaction-level data.
- Engineer customer-level behavior features.
- Apply K-Means Clustering for customer segmentation.
- Evaluate the clustering result using inertia, silhouette score, Davies-Bouldin score, cluster size balance, and business interpretability.
- Translate each segment into practical business recommendations.

## Dataset

Dataset: Olist Brazilian E-Commerce Public Dataset  
Source: <https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce>

The dataset is public and anonymized. It is used only for learning, portfolio, and exploratory analysis.

## Main Method

K-Means Clustering.
