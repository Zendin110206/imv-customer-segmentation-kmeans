# Project Notes

## Project Scope

This project focuses on customer segmentation using K-Means Clustering.

The analysis starts from raw Olist e-commerce tables, builds customer-level behavior features, applies scaling, compares several cluster counts, and interprets the final customer segments.

## Current Decision

Dataset selected: Olist Brazilian E-Commerce Public Dataset.

Reason:

- It is richer than simple toy datasets.
- It supports customer behavior analysis.
- It includes transactions, payments, reviews, delivery timestamps, and product information.
- It is still realistic for an IMV individual assignment.

## Important Claim Boundary

The final recommendations are exploratory and should be treated as decision-support insights. They are not proof of actual business impact because no A/B testing or real campaign validation is performed.
